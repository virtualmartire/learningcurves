Welcome to LearningCurves

LearningCurves is an ultralight, cross-platform, 100% privacy-preserving experiments tracking app for machine learning:
  • no installation required, runs in the browser in pure javascript;
  • accessible from every online device;
  • computation managed by the frontend only, zero data sent to the cloud.
For requests and suggestions please contact the project manager Stefano Martire.

Just load a run to start using the app!
